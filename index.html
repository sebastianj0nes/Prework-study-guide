HTML

  • The head element contains information about the page
  • Inside the head will contain metadata on how to display the website
  • The body element represents the visible content shown to the webpage
  • Inside the body many different elements can be used, such as these unordered lists

CSS

  • CSS = Casscading Styling sheets
  • CSS is used to style a page and its elements
  • Invidiual elements can be styled, or groups using "div" or "section"
  • 3 types of styling; inline css, in-page css and external css

Git

  • Git commands (in CLI) helps with version control;
  • git status - will check which branch we are in
  • git add -A - stage changes before comitting
  • git checkout -b 'branch-name' creates a new branch
  • git commit -m 'message' commits files to repository with an intended message
  • git pull origin main - syncing local branch with branch in GitHub to make sure code is up to date
  • git push origin 'branch-name' will push files to the specified branch

Javascript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code -running from top to bottom
  • Logic statements -e.g 'if' & 'for' loops can help not write out multiple lines of code
  • Functions - can be created and called upon; which can include logic and conditional statements
